COVID-19 Case Rates Across Canadian Health Regions

Background

This report was developed to highlight the current COVID-19 situation across Canada. Providing case rate statistics by local health regions is better suited to detect outbreaks or local health concerns. The case rate used here is the count of new cases reported in the previous 7 day, divided by 100,000 population.

This report was completed using RMarkdown and was completed as part of a web scraping, automation, and visualization project. COVID-19 data is automatically scraped, processed, visualized, and uploaded everyday at ____.

Provincial and territorial COVID-19 data was imported from Canada.ca, and health region data was imported from COVID-19 Canada. Note that some regions report data at different frequencies, so case rates may be a day or two delayed in a few instances.

The ggplot2 package was used to display the provincial and territorial case rate, and the reactable package was used to develop the interactive table.
